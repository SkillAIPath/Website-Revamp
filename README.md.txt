# SkillAI Path Website

AI & Data Science Career Training Platform

## Pages
- Homepage with hero, course preview, success stories
- Courses page with detailed track information
- Success stories with student testimonials
- Contact form and information
- FAQ with expandable sections
- Enrollment form with track selection
- Strategy call booking page
- Legal pages (Privacy, Refund, Terms)

## Features
- Responsive design for all devices
- Interactive elements (FAQ accordion, form validation)
- Consistent branding and navigation
- Professional contact and enrollment forms

## Deployment
- Optimized for Netlify deployment
- No external dependencies except Google Fonts
- All assets self-contained

Built with HTML, CSS, and vanilla JavaScript.
